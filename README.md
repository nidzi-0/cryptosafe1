# CryptoSafe Manager

## 1. Видение проекта

CryptoSafe Manager - это кроссплатформенный менеджер паролей с локальным зашифрованным хранилищем, построенный на модульной архитектуре с возможностью поэтапного расширения.

Основные принципы проекта:

- Безопасное хранение данных (шифрование на уровне приложения)
- Отсутствие жёстко закодированных секретов
- Чёткое разделение модулей (архитектура MVC)
- Расширяемость (аудит, резервное копирование, импорт/экспорт)
- Поддержка CI и тестирования с первого спринта

Проект реализуется по модели поэтапной разработки (8 спринтов).

---

## 2. Дорожная карта спринтов

### Sprint 1 — Foundation (текущий)
- Архитектура проекта (MVC)
- SQLite схема базы данных
- Абстрактный криптосервис
- XOR-заглушка шифрования
- EventBus (pub/sub)
- GUI оболочка
- Базовые тесты
- GitHub Actions

### Sprint 2 — Key Management
- Реальное управление мастер-ключом
- Хранение параметров в key_store
- Интеграция KeyManager

### Sprint 3 — Real Encryption
- Замена XOR на AES-GCM
- Аутентифицированное шифрование

### Sprint 4 — Clipboard Security
- Безопасное копирование
- Авто-очистка буфера обмена

### Sprint 5 — Audit Signatures
- Цифровая подпись записей аудита
- Проверка целостности

### Sprint 6 — Import/Export
- Экспорт в защищённый формат
- Импорт существующих данных

### Sprint 7 — Auto-Lock & Session Control
- Таймер неактивности
- Автоматическая блокировка

### Sprint 8 — Backup & Packaging
- Резервное копирование
- Dockerfile
- Финальная сборка
